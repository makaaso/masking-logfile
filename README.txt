# masking-logfile

## 機能
- ログファイルの特定箇所をマスキングする
- 個人情報などのマスキングをする時に便利
- sedコマンドで正規表現を使って置換

## 使い方

```
git clone https://github.com/makaaso/masking-logfile.git
cd masking-logfile
./masking-logfile.sh
```

## 補足
- 実際に利用するときはmasking-logfile.shの「sample*」を適当なディレクトリに変更
- ファイルが圧縮されていない場合は、ファイル解凍のところをコメントアウト
